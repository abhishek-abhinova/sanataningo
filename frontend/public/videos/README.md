# Videos Directory

This directory contains video files for the gallery section.

## How to Add Videos

1. **Video Files**: Place your MP4 video files in this directory with the following names:
   - `community-service.mp4` - Community service activities
   - `cultural-event.mp4` - Cultural events and festivals
   - `educational-program.mp4` - Educational initiatives
   - `spiritual-gathering.mp4` - Spiritual sessions and gatherings
   - `featured-documentary.mp4` - Main featured video

2. **Video Poster Images**: Add corresponding poster images in the `/public/images/` directory:
   - `video-poster1.jpg` - Poster for community service video
   - `video-poster2.jpg` - Poster for cultural event video
   - `video-poster3.jpg` - Poster for educational program video
   - `video-poster4.jpg` - Poster for spiritual gathering video
   - `featured-video-poster.jpg` - Poster for featured documentary

## Video Requirements

- **Format**: MP4 (recommended for best browser compatibility)
- **Resolution**: 1280x720 (HD) or higher
- **File Size**: Keep under 50MB for better loading performance
- **Audio**: Include audio tracks for full experience
- **Duration**: 3-10 minutes recommended for web viewing

## Video Controls

The gallery includes:
- ✅ Play/Pause controls
- ✅ Volume control with sound
- ✅ Fullscreen option
- ✅ Progress bar
- ✅ Duration display
- ✅ Poster image preview

## Fallback

If video files are not available, the gallery will show placeholder content with "Video Coming Soon" message.
