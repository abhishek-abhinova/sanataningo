# Upload Folders

This directory contains uploaded files from the admin dashboard:

- **gallery/** - Gallery images uploaded by admin
- **aadhaar/** - Member Aadhaar document images  
- **screenshots/** - Payment screenshot images
- **team/** - Team member profile photos
- **videos/** - Uploaded video files

## Note:
These folders serve as placeholders. In production, files should be uploaded to:
`https://sarboshaktisonatanisangathan.org/uploads/`

## Folder Structure:
```
uploads/
├── gallery/
├── aadhaar/
├── screenshots/
├── team/
└── videos/
```
